        # 1 换源

        • 错误 下载元数据失败：Cannot prepare internal mirrorlist: No URLs in mirrorlist
        • 解决 直接运行命令sudo sed -i -e "s|#baseurl=http://mirror.centos.org|baseurl=http://vault.centos.org|g" /etc/yum.repos.d/CentOS-*更换源
